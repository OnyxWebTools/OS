How to enable more Onyx OS features

NOT COMPATIBLE WITH INTERNET EXPLORER, SAFARI, MOBILE BROWSERS

  1. Download the file (Download)
  2. Unzip the file to a location such as your userfolder / home directory
  3. Name it “OnyxOS Addon DO NOT DELETE”
  4. In your browser’s addressbar type “chrome://extensions”
  5. Find the “Developer Mode” toggle (left column / top right), turn it to on
  6. Press load unpacked (somewhere at the top)
  7. Locate unzipped folder “OnyxOS Addon DO NOT DELETE”
  8. Select it (1 click)
  9. Press OK
  10. Pin it (Puzzle piece on browser toolbar, then the pin/eye next to OnyxOS)
  11. Tap it to open OnyxOS
